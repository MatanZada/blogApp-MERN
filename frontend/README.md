## Installation

Install the dependencies and devDependencies

[Node.js](https://nodejs.org/)

```sh
cd /server
npm install
npm run dev
```

[React.js](https://reactjs.org/)

```sh
cd /client
npm install
npm start
```

# About this Project

Apply, Build a Blog Project using Redux (Redux Toolkit), NodeJs, Express, MongoDB, React js, Tailwindcss
